\chapter{Objetivos}

\section{Objetivo principal}
Diseñar y desarrollar un prototipo de robot de compañía móvil, integrado con un modelo de inteligencia artificial conversacional, capaz de interactuar de forma autónoma y asistir al usuario dentro de un entorno doméstico.

\section{Objetivos secundarios}
Se plantean los siguientes objetivos secundarios:

\begin{enumerate}
    \item Desarrollar el sistema de movimiento autónomo del robot, incluyendo el diseño del chasis, la integración de motores y la programación de un sistema de evitación de obstáculos, utilizando un microcontrolador \textbf{Arduino Uno}.
    \item Implementar inteligencia artificial en la plataforma \textbf{Raspberry Pi}, abarcando la configuración del sistema, la creación de una API como intermediaria entre el host del LLM y la \textbf{Raspberry Pi}, y la integración de modelos de reconocimiento de voz (speech-to-text).
    \item Integrar periféricos de comunicación (entrada y salida) para la interacción usuario-robot, incluyendo la instalación de micrófono y altavoz.
    \item Ensamblar los distintos módulos de hardware del prototipo, procurando la coordinación entre \textbf{Arduino} y \textbf{Raspberry Pi}.
    \item Validar la funcionalidad completa del robot mediante pruebas de software y hardware.
\end{enumerate}
